## 🚀 How to Use

1. Clone this repository:

   ```bash
   git clone https://github.com/Akshay7368/python_work.git
   ```
2. Navigate into the folder:

   ```bash
   cd python_work
   ```
3. Open Jupyter Notebook:

   ```bash
   jupyter notebook
   ```
4. Explore the project files.
